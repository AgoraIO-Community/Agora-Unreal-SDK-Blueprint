◆ enumerateVideoDevices()

VideoDeviceCollection* agora::rtc::ue4::AgoraVideoDeviceManager::enumerateVideoDevices ( )

Enumerates the video devices.

This method returns an VideoDeviceCollection object including all video devices in the system. With the VideoDeviceCollection object, the application can enumerate the video devices.

Note
The application must call the release method to release the returned object after using it.
Returns
  • An IVideoDeviceCollection object including all video devices in the system: Success.
  • nullptr: Failure.
